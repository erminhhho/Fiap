Como testar:

  1. Selecione "Instituidor" em qualquer um dos campos de relacionamento
  2. Observe que o label "Nascimento" deve mudar automaticamente para "Falecimento"
  3. Mude para outra opção e veja que volta para "Nascimento"
  4. Teste em ambos os autores para verificar se funciona independentemente
  5. Verifique no log se as classes CSS foram aplicadas corretamente

Melhorias implementadas:

  • Múltiplas estratégias de busca do label
  • Compatibilidade com labels flutuantes (input-label)
  • Classes CSS para identificação (birth-label/death-label)
  • Logs detalhados para debugging
  • Preservação de placeholders originais
